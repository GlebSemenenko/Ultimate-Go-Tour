Основные пакеты стандартной библиотеки Go

1. Основные пакеты
    fmt — форматированный ввод-вывод (печать, сканирование)
    io и io/ioutil — работа с потоками и файлами
    os — взаимодействие с операционной системой (файлы, переменные окружения)
    errors — создание и обработка ошибок
    log — логирование
    time — работа со временем и датами
    strconv — преобразование строк и чисел
    strings — работа со строками
    runtime - низкоуровневые функции для работы с внутренним механизмом выполнения программы. 
    Она обеспечивает доступ к информации о системе, управлению памятью, 
    работе с горутинами, обработке ошибок и другим важным аспектам выполнения программы.
    reflect - рефлексия

2. Работа с коллекциями
    array — массивы фиксированной длины
    slice — срезы, динамические массивы
    map — ассоциативные массивы (словарь)

3. Работа с файлами и потоками
    os — создание, удаление, чтение файлов
    bufio — буферизация ввода-вывода
    io — интерфейсы для чтения и записи

4. Конкурентность
    sync — мьютексы, группы ожидания (WaitGroup)
    context — управление временем жизни операций

5. Работа с сетью
    net — создание TCP/UDP серверов и клиентов
    http — создание HTTP серверов и клиентов

6. Работа с JSON и другими форматами
    encoding/json — сериализация и десериализация JSON
    encoding/xml — работа с XML

7. Стандартные утилиты
    flag — парсинг командной строки
    math — математические функции
    regexp — регулярные выражения
